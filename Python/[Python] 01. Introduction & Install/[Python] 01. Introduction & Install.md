### Python 3.8.9 다운로드

(윈도우 7 이상 환경)

1. <https://www.python.org/downloads/release/python-389/> 접속

2. 스크롤 내려 Files - Windows installer (64-bit) 클릭

   ![1](1.png)

3. 설치 실행

4. Add Python 3.8 to PATH 체크 후 Install Now

   ![2](2.png)

5. Disable path length limit 클릭

   ![3](3.png)

6. 완료
